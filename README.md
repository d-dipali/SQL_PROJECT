📊 SQL Data Cleaning Project: Global Layoffs Dataset
📌 Project Overview :
This project focuses on cleaning and preparing a global layoffs dataset using SQL. The dataset contains information about layoffs across various companies, industries, and countries. The goal is to transform raw, inconsistent data into a structured and reliable format suitable for analysis.
Through this project, key data cleaning techniques such as handling missing values, removing duplicates, standardizing data formats, and correcting inconsistencies are applied to ensure high-quality data.

🎯 Objectives :
Clean and preprocess raw layoff data using SQL, 
Identify and remove duplicate records,
Handle null and missing values effectively,
Standardize inconsistent data (e.g., company names, industries, dates),
Prepare the dataset for exploratory data analysis (EDA),
Ensure data accuracy and integrity for reliable insights.

🛠️ Tools & Technologies :
SQL (MySQL / PostgreSQL / SQL Server),
Relational Databases,
Data Cleaning Techniques.

📂 Dataset Description :
The dataset includes the following key fields:
Company Name,
Location,
Industry,
Total Laid Off,
Percentage Laid Off,
Date,
Stage (Startup stage, etc.),
Country,
Funds Raised.

🔍 Data Cleaning Steps :
Removed duplicate records using window functions,
Standardized text fields (company names, industries),
Converted date formats into a consistent structure,
Handled null and blank values,
Trimmed extra spaces and corrected inconsistencies,
Verified data types and ensured column integrity.

📈 Key Outcomes :
Created a clean and analysis-ready dataset,
Improved data consistency and reliability,
Reduced errors caused by duplicate and missing data,
Established a strong foundation for further data analysis and visualization.

🚀 Future Scope :
Perform exploratory data analysis (EDA),
Build visual dashboards using tools like Tableau or Power BI,
Analyze layoff trends by industry, country, and time,
Derive business insights and patterns.

This project is for educational and portfolio purposes, using publicly available layoff datasets.
